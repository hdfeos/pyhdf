================
Installing pyhdf
================

``pyhdf`` has been successfully installed under Python 2.7 and above on
several platforms, including 

* Windows 7

* Linux 2.6

* Mac OS X Mavericks

Please open an issue here if you encounter any problems during installation:
http://hdfeos.org/forum
